# Excel Data Validator

Dự án Python để kiểm tra và validate dữ liệu trong file Excel.

## Tính năng
- Đọc file Excel 
- Thêm cột validation check
- Xuất file Excel với kết quả kiểm tra

## Cài đặt
```bash
pip install -r requirements.txt
```

## Sử dụng
```python
python excel_validator.py
```

## Yêu cầu
- Python 3.7+
- pandas
- openpyxl
- xlsxwriter
